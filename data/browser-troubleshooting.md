# Browser loading and cache troubleshooting

## Quick recovery

First check the Lumina status page. Then open a private browsing window. If the page works privately, clear site data for the Lumina domain only, close all Lumina tabs, and sign in again. Supported browsers are the latest two stable versions of Chrome, Edge, Firefox, and Safari.

## Persistent loading spinner

Disable content blockers for the Lumina domain and allow third-party cookies for single sign-on. Corporate networks must permit WebSocket traffic over port 443. If the problem remains, capture the browser name and version, timestamp, console errors, and the request ID shown in Settings > Diagnostics.
