# Account security and access recovery

Support can guide a password reset but cannot view passwords, disable multi-factor authentication, change account ownership, or disclose workspace data without identity verification. Password reset links expire after 30 minutes and can be used once.

If an account may be compromised, revoke active sessions, rotate API keys, reset the password, and review audit logs. Requests to change an owner email, remove MFA, or delete an account must be escalated to the account security team. Never collect passwords or one-time codes in support chat.
